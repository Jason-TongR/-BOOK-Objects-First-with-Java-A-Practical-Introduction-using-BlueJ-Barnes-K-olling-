Project: calculator-full-solution-FXML
Authors: Michael Kölling and David J. Barnes

This project is part of the material for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Seventh edition
   David J. Barnes and Michael Kölling


This version of the project implements a complete solution to the simple 
calculator discussed in Chapter 9. It implements the GUI using FXML (built 
with an external GUI builder.)

Start this application by choosing 'Run JavaFX Application' from the class menu.
