Project: recursion-experiments
Authors: David J. Barnes and Michael Kölling

This project contins some simple test classes and methods to do some
initial experiments with recursive methods. It is part of the material 
for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Seventh edition
   David J. Barnes and Michael Kölling
   Pearson Education

It is discussed in chapter 11.
