Project: musicplayer (JavaFX version) 

A project to play audio files.
Authors: David J. Barnes and Michael Kölling

This project is part of the material for the book

   Objects First with Java - A Practical Introduction using BlueJ
   Seventh edition
   David J. Barnes and Michael Kölling

It is discussed in chapter 17.

To run this project, use the 'Run JavaFX Application' option from the main class's menu.
For details, refer to the book chapter.
